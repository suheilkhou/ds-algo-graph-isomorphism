# DS&A — Graphs: Isomorphism, Complements, Connectivity

## About the course

This repository is part of my coursework for **Data Structures and Algorithms**, a university course on algorithm analysis and design. Topics include asymptotic analysis, sorting, balanced search trees, hashing, graphs and shortest-path algorithms, dynamic programming, and amortized analysis — with both written theoretical exercises and a final programming project.

## About this assignment

The exercise focuses on structural properties of graphs: isomorphism between simple graphs, complementary graphs, and classical connectivity claims. A representative question asks to prove that if an undirected graph `G` is not connected then its complement `G̅` must be connected. Answers combine standard definitions with direct proofs and, where the claim fails, small concrete counter-examples.
