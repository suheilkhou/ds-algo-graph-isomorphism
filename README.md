# DS&A — Graph Theory Basics

## About the course

This repository is part of my coursework for **Data Structures and Algorithms**, a university course on algorithm analysis and design. Topics include asymptotic analysis, sorting, balanced search trees, hashing, graphs and shortest-path algorithms, dynamic programming, and amortized analysis — with both written theoretical exercises and a final programming project.

## Topic

Graphs: isomorphism, complementary graphs, and connectivity.
